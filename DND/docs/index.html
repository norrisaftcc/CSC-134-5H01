About the Game

DND Text Adventure is a text-based role-playing game that brings the classic tabletop experience to your terminal. Journey through a fantasy world, make choices that affect your destiny, and battle enemies using a simplified D&D ruleset.

The game includes character creation, leveling, combat mechanics, and story progression. You can save your progress and resume your adventure at any time.

Getting Started

  1. Compile the game using: g++ -o main_game main_game.cpp -std=c++11
  2. Run the game with: ./main_game
  3. Follow the in-game prompts to create a character or load a saved one
  4. Embark on your adventure!

For more detailed instructions and game mechanics, check out the Documentation section.
