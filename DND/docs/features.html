Game Features

Discover all the exciting features of the DND Text Adventure

Character Creation and Development

👤

Character Creation

Create unique characters by selecting from seven different classes: Fighter, Magic-User, Thief, Cleric, Elf, Dwarf, and Halfling.

Roll dice to determine your starting attributes: Strength, Dexterity, Constitution, Intelligence, Wisdom, and Charisma.

🔄

Save & Load

Save your character's progress at any time and resume your adventure later.

Secure password system protects your character data with encryption.

📈

Character Progression

Gain experience points for overcoming challenges and defeating enemies.

Level up and earn stat points to improve your character's abilities.

Gameplay Mechanics

🎮

Interactive Storytelling

Explore a branching narrative with meaningful choices.

Progress through the story and unlock different paths and endings.

⚔️

Combat System

Engage in turn-based combat using a simplified D&D ruleset.

Roll dice to determine attack success, with bonuses based on your character's class and attributes.

🎭

Class-Specific Features

Each class has unique bonuses and abilities in combat.

Experience different gameplay styles depending on your chosen class.

Technical Features

🔒

Secure Data Storage

Character data is encrypted for security.

Password-protected character access ensures only you can play your character.

💾

Persistent Game State

The game remembers your progress and choices.

Resume exactly where you left off in the story.

🔧

Testing & Debugging Tools

Built-in tools for creating and testing characters.

Toggle saving feature for testing different scenarios without affecting save files.

Recent Improvements

  • Fixed Leveling System: Experience points are now properly accumulated and not reset after leveling up
  • Enhanced Character Creation: Improved character stats saving and loading
  • Stat Points Calculation: Stat points are correctly awarded when leveling up and tracked properly
  • Save Toggle Feature: Added ability to disable saving for testing purposes
  • Testing Tools: Comprehensive test suite for verifying gameplay mechanics
